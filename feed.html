The Feed


In my recent web design project, "The Feed," I crafted a social media platform following a given rubric. This was my very first project in CSS and HTML, so it was a big step for me. The homepage featured a dynamic GIF logo, creating an engaging visual experience. I implemented a friends list feature and integrated links to my favorite websites, adding a personalized touch. The project also included an informational page explaining the platform's features and a splash page to capture users' attention. The central feed page displayed media seamlessly, ensuring a user-friendly experience. This project showcased my ability to balance creativity and functionality, resulting in a polished and professional social media platform.
