Project Overview

This project focused on creating a series of web banners for an online service as part of my class assignment. The goal was to design eye-catching banners that not only grab attention but also communicate the essence of the service clearly and effectively. Each banner is crafted to enhance user engagement and ensure a seamless experience for visitors.

Design Approach
  • User-Centered Design: Prioritized clarity and visual appeal to ensure users get the message at a glance.
  • Adaptable Layouts: Designed with flexibility in mind to work across different screen sizes and formats.
  • Brand Alignment: Each banner was developed to align with the brand's tone, using consistent colors, fonts, and visual elements.
